Simple Order ML pipeline
Files:
 models/simple_order_model.joblib - trained model
 data_simple_orders.csv - sample data used
 test_predictions.csv - test set with predictions
 metrics.json - accuracy and report
 requirements.txt - pip install requirements

To serve model:
 1) pip install -r requirements.txt
 2) use the serve_simple.py provided earlier to serve via FastAPI
